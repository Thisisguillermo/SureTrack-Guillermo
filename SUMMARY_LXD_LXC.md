# LXD/LXC (https://linuxcontainers.org/lxd/getting-started-cli/)

```
sudo dnf install snapd

sudo ln -s /var/lib/snapd/snap /snap

sudo systemctl restart snapd.service

sudo snap install lxd

sudo snap enable lxd

sudo snap start lxd
```
